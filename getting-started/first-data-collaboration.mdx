In this tutorial, you'll set up your first data collaboration. Collaborations let you share data with partners, customers, or other organizations while maintaining control over access and privacy.

</Info>

## Prerequisites

- A Narrative I/O account
- At least one normalized dataset
- A collaboration partner (or use the demo partner for testing)

## What you'll learn

- How to create a data collaboration
- How to set access controls and permissions
- How to share data securely with a partner
